# 🌱 Os Impactos do Agronegócio no Meio Ambiente

Site educativo desenvolvido para apresentar os principais impactos do agronegócio sobre o meio ambiente, destacando problemas ambientais e possíveis soluções sustentáveis.

🔗 Acesse o projeto: https://bianca043.github.io/Os-Impactos-do-Agronegocio-no-meio-ambiente/

## 📖 Sobre o projeto

O agronegócio desempenha um papel fundamental na economia brasileira, representando uma parcela significativa do PIB e das exportações. Entretanto, sua expansão também gera consequências ambientais importantes.

Este site foi criado com o objetivo de informar e conscientizar sobre:

- Desmatamento;
- Uso excessivo de agrotóxicos;
- Poluição da água;
- Emissão de gases de efeito estufa;
- Perda da biodiversidade;
- Degradação do solo.

Além disso, apresenta alternativas sustentáveis para reduzir esses impactos. :contentReference[oaicite:0]{index=0}
